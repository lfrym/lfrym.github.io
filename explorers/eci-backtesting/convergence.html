ECI Forecast Convergence

How do ECI-based benchmark forecasts converge to actual SOTA as the prediction window shrinks? Target date: 2026-02-27

Convergence Backtesting

Instead of forecasting different horizons from each date, we fix a single target date (2026-02-27) and predict it from 12, 11, 10, …, 1 months prior. As the prediction window shrinks, we expect the forecast to converge to the actual value.

Methodology

For each lookback period (N months before target):

  • Filter data to models released before that date
  • Fit an ECI model using IRT
  • Extract trend data for top-N models by ECI at release
  • Fit a trend using BIC model selection or Last-M-Months regression
  • Forecast ECI forward N months and convert to benchmark scores
  • Compare against actual SOTA at the target date

Trend Methods

  • BIC: Compares linear vs piecewise (single breakpoint) models using Bayesian Information Criterion.
  • Last M Months: Simple linear regression on only the last M months, where M equals the lookback period.
